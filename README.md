# BrailleToKor_Python

점자 문자열을 한국어로 번역할 수 있는 BrailleToKor 파이썬 패키지입니다.

BrailleToKor Python package that translates braille string to Korean.

## Usage Example
```python
from BrailleToKorean.BrailleToKor import BrailleToKor

b = BrailleToKor()
print(b.translation("⠣⠒⠉⠻⠚⠠⠝⠬⠲⠀⠨⠎⠢⠨⠀⠘⠾⠱⠁⠀⠙⠗⠋⠕⠨⠕⠕⠃⠉⠕⠊⠲"))
```
